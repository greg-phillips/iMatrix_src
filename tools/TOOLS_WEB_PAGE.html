1. Install Dependencies

2. Get API Access

Your iMatrix Cloud email and password

3. Run Tools

üîç Explore Your Data

Complete interactive workflow requiring no prior knowledge:

cd tools/get_sensor_data
python3 get_sensor_data.py -u your@email.com
Discovers devices ‚Üí sensors ‚Üí prompts for dates ‚Üí downloads data
üó∫Ô∏è Get Location Tracks

Download GPS history as GeoJSON for Google Maps:

cd tools/get_loc.
python3 get_loc.py -s 1234567890 -ts 1736899200000 -te 1736985600000 -u your@email.com
Creates {serial}-{timestamp}.geojson file

Tool Selection Guide

I want to...

Use get_sensor_data.py - Interactive device and sensor discovery
python3 get_sensor_data.py -u your@email.com

Use get_loc.py - Downloads lat/lon and creates GeoJSON
python3 get_loc.py -s {serial} -ts {start} -te {end} -u your@email.com

Use upload_tsd.py - CSV to iMatrix Cloud with validation
python3 upload_tsd.py -U your@email.com -G {gateway} -C {controller} -D data.csv

Use link_sensor.py - Individual or batch sensor linking
python3 link_sensor.py -u your@email.com -pid {product_id} -sid {sensor_id}

API Reference

üîó Common API Endpoints

Authentication
POST /login
Used by: All tools
Device Management
GET /things/
Used by: get_sensor_data.py
GET /products/{id}
Used by: get_sensor_data.py
Sensor Management
GET /sensors/{id}
Used by: get_sensor_data.py
POST /sensors/predefined
Used by: upload_internal_sensor.py
POST /sensors/predefined/link
Used by: link_sensor.py
Data Access
GET /things/{serial}/sensor/{id}/history/{start}/{end}
Used by: get_sensor_data.py, get_loc.py
File Management
POST /files
Used by: upload_image.py
Data Upload
POST /tsd/upload
Used by: upload_tsd.py, upload_sine_wave.py
API Environments
Development: api-dev.imatrixsys.com
Production: api.imatrixsys.com
